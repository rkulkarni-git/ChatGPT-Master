# Skill: FAQ Build Framework

## Objective

Define standardized method to create FAQ documents.

---

## Structure

```text
# FAQ: <topic>

## Q1. <question>
Answer:
<clear answer>

## Q2. <question>
Answer:
<clear answer>
```

---

## Rules

* Questions must be real-world
* Answers must be concise
* No duplication
* No procedural steps (belongs to SOP)

---

## Version Control

* Maintain version in metadata
* Maintain changelog
