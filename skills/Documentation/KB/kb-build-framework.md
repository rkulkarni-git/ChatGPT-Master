# Skill: KB (Knowledge Base) Build Framework

## Objective

Define standardized method to document knowledge and explanations.

---

## Structure

```text
# KB: <topic>

## Overview
<explanation>

## Concepts
<technical explanation>

## Use Cases
<real-world application>

## References
<optional>
```

---

## Rules

* Explanatory, not procedural
* Deep technical clarity
* No step-by-step (belongs to SOP)
* No Q&A (belongs to FAQ)
