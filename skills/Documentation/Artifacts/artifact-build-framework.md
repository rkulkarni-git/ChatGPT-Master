# Skill: Artifact Build Framework

## Objective

Define standardized method to capture full conversation history.

---

## Structure

```text
# Artifact: <project>

### Entry 1
User Prompt:
<exact>

ChatGPT Response:
<exact>

### Entry 2
...
```

---

## Rules (CRITICAL)

* Append-only
* No edits
* No summarization
* No SOP formatting
* No transformation

---

## Scope

* Entire chat history
* All prompts and responses
